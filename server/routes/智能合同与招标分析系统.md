- `mammoth` 库尝试将 `.doc` 文件当作ZIP文件解析，导致报错：`Can't find end of central directory : is this a zip file ?`
- 原代码没有区分 `.doc` 和 `.docx`，也没有清晰的错误提示

2. **明确错误提示**：对 `.doc` 格式抛出友好错误："不支持旧版 .doc 格式，请将文件另存为 .docx 格式后再上传"

- **任务分工**: 自动生成投标任务分配方案，包含负责人和截止日期
